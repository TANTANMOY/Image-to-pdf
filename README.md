
# Image to Pdf 

From this React base webApp you can generate image to pdf file.



## Demo
https://image2pdf.netlify.app/



## Run Locally

Clone the project

```bash
 Clone this repo
```
Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
To Visit App

```bash
  Open http://localhost:3000 to view it in the browser.
```
